# TNTFS

MacOS M1 开源NTFS

Python3.10 使用 py2app 打包编译

理论上 MacOS 全系列通用

#### 安装 macfuse

关闭SIP保护特性

系统中启用 macfuse

```shell
brew install ntfs-3g-mac
brew install macfuse
```

#### 运行

把软件放到应用程序里然后双击运行
